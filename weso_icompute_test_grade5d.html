Programming Task — Weather Station Reporter
Scratch 3.29 offline

Create a sprite that acts as a weather station. The sprite collects 5 days of temperature readings using a loop, tracks how many days were hot, and then gives a summary report. Your program must do all of the following:

1
When the green flag is clicked, play a sound. Then create four variables: day, temp, total, and hot_days. Set all four to 0. Make temp and hot_days visible on the stage.
2
Use a repeat 5 loop. Inside the loop each time:
  • Change day by 1
  • Ask "Enter today's temperature (°F):" and set temp to the answer
  • Change total by temp (keep a running total of all temperatures)
  • If temp > 80: change hot_days by 1
3
After the loop, use if / else to check whether there were more than 2 hot days:
  • If hot_days > 2: switch to a sunny backdrop and say (join "Hot week! " (join hot_days " days were over 80°F.")) for 3 seconds
  • Else: switch to a cloudy backdrop and say (join "Mild week. Hot days: " hot_days) for 3 seconds
4
Make sure temp and hot_days are visible on stage so the coach can watch values update as each day is entered.

Precision (temp & hot_days visible; join used in both output messages; boundary strictly >80 not ≥; messages shown for exactly 3 seconds)5 pts

If you won, you earn bonus points equal to the current round number — so winning round 1 is worth 1 pt, round 2 is worth 2 pts, round 3 worth 3 pts, and round 4 worth 4 pts. The algorithm then prints your total bonus at the end.

3 pts: mentions variable bonus point value per round, the 4-round loop, and the final print. 2 pts: missing one element. 1 pt: partial understanding.

Loop 2: round = 2, result = "no" → no change → bonus = 1
Loop 3: round = 3, result = "yes" → bonus += 3 → bonus = 4
