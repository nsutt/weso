Programming Task — Snack Shop Calculator
Scratch 3.29 offline

Create a sprite that acts as a snack shop register. The sprite asks what the customer wants to order, calculates the total cost using variables, and tells them whether they went over budget. Your program must do all of the following:

1
When the green flag is clicked, create three variables: cookies, juice, and total. Make cookies and juice visible on the stage.
2
Ask "How many cookies would you like?" and set cookies to the answer. Then ask "How many juice boxes would you like?" and set juice to the answer.
3
Set total using the cost formula — cookies cost 2 coins each and juice boxes cost 3 coins each:
  • total = (cookies × 2) + (juice × 3)
4
Use if / else to check the total:
  • If total > 10: say "You went over budget! Cost: [total] coins." for 3 seconds
  • Else: say "Your order costs [total] coins. Enjoy!" for 3 seconds
Use the join block to include the value of total inside each message.
5
Switch the backdrop to match the result — use a different backdrop for over-budget vs. in-budget.
6
Add a sound that plays when the program starts.

Precision (exact formula, join embeds total in both messages, correct >10 boundary, 3-second say)5 pts
