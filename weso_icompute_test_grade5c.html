Programming Task — Star Catcher Game
Scratch 3.29 offline · 2 sprites

Build a game where the player moves a catcher sprite left and right to catch falling stars. The game ends when the player misses 3 stars. You will need two sprites — a Catcher and a Star — each with their own scripts.

1
Create two variables (both visible on stage): score (set to 0) and misses (set to 0).
2
Catcher sprite — add these scripts so the player can move it:
Catcher scripts
  • When right arrow key pressed → change x by 15
  • When left arrow key pressed → change x by −15
3
Star sprite — add a script that starts when the green flag is clicked:
Star script
  • Set position to x: pick random −200 to 200, y: 170
  • Start a forever loop:
  • — Change y by −8 (star falls down)
  • — If touching Catcher: change score by 1 → play a sound → go to x: pick random −200 to 200, y: 170
  • — If y < −160 (hit the floor): change misses by 1 →
    • If misses = 3: say (join "Game Over! Score: " score) for 3 seconds → stop all
    • Else: go to x: pick random −200 to 200, y: 170
4
Add a backdrop to the stage. Optionally, add a costume change on the Catcher or Star when a catch is made.
Scoring rubric — 30 pts total
Correct algorithm (catcher moves with arrow keys; star falls in forever loop; collision detection; misses increments on floor; game stops at misses = 3)10 pts
Working game (catcher moves, star falls and resets, score goes up on catch, game ends correctly at 3 misses)10 pts
Precision (score and misses both visible; join used in game-over message; star resets to random x each time; stop all used at game over)5 pts
Creativity & efficiency — tiebreaker (sound on catch, backdrop, costumes, increasing speed or other enhancements)5 pts
