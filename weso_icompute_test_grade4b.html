Programming Task — Quiz Score Grader
Scratch 3.29 offline

Create a program where a sprite acts as a grade calculator. The sprite asks for a quiz score and tells the student what letter grade they earned. Your program must do all of the following:

1
When the green flag is clicked, have a sprite ask the user for their quiz score (a number from 0 to 10).
2
Based on the score entered, the sprite should say the matching grade message for 3 seconds:
ScoreMessage to say
9 or 10"Excellent — that's an A!"
7 or 8"Great job — that's a B!"
5 or 6"Good try — that's a C."
0 to 4"Keep studying! You'll do better next time."
3
Change the sprite's costume after saying the grade — use at least two different costumes (e.g. happy face for A/B, thinking face for C/below).
4
Change the backdrop to match the result — use a different backdrop for a passing grade (5 or above) and a failing grade (below 5).
5
Add a sound that plays when the program starts.
Scoring rubric — 30 pts total
Correct algorithm (if/else logic correctly covers all four score ranges)10 pts
Working program (ask works, correct grade message appears for each range)10 pts
Precision (exact message text, correct score boundaries, 3-second say)5 pts
Creativity & efficiency — tiebreaker (costumes, backdrops, sound, clean code)5 pts
