Programming Task — Activity Suggester
Scratch 3.29 offline

Create a program that asks the user for a temperature and suggests an outdoor activity. Your program must do all of the following:

1
When the green flag is clicked, have a sprite ask the user for a number between 0 and 100. This number represents the temperature outside.
2
Based on the temperature entered, the sprite should say the matching activity for 3 seconds, using the table below:
TemperatureActivity to suggest
0 – 39"Time to go skiing!"
40 – 59"Great day for golf!"
60 – 79"Let's play tennis!"
80 – 100"Perfect for swimming!"
3
Switch the backdrop to match each temperature range — use a different backdrop for each of the four activities.
4
Add background music that plays when the program starts.
Scoring rubric — 30 pts total
Correct algorithm (if/else logic covers all 4 ranges correctly)10 pts
Working program (ask works, correct activity appears for each input)10 pts
Precision (exact message text, correct temperature boundaries)5 pts
Creativity & efficiency — tiebreaker (backdrops, music, costume changes)5 pts
