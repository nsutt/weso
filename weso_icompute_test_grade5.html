Programming Task — Number Guessing Game
Scratch 3.29 offline

Create a number guessing game where the player tries to discover a secret random number. Your program must do all of the following:

1
When the green flag is clicked, pick a random number between 1 and 20 and store it in a variable called secret.
2
Create a variable called guesses, set it to 0 at the start, and make it visible on the screen.
3
Use a loop that keeps asking the player to guess a number. The loop must stop when the player guesses correctly or when they have used all 5 guesses.
4
Each time the player makes a guess, add 1 to guesses, then respond:
  • Guess is too high → say "Too high! Try again." for 2 seconds
  • Guess is too low → say "Too low! Try again." for 2 seconds
  • Guess is correct → say "You got it!" for 3 seconds, then stop
5
If the player uses all 5 guesses without getting it right, say "Out of guesses! The number was [secret]." — use the join block to include the secret number in the message.
6
Add a backdrop to the stage. For bonus points: add a sound when the player wins or loses.

Correct algorithm (loop, variables, if/else logic)10 pts
Working program (runs correctly without freezing)10 pts
Precision (exact messages, correct bounds, guesses counter works)5 pts
Creativity & efficiency — tiebreaker (backdrop, sounds, clean code)5 pts
