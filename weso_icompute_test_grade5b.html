Programming Task — Coin Flip Counter
Scratch 3.29 offline

Create a program that simulates flipping a coin 10 times, counts the results, and announces the winner. Your program must do all of the following:

1
Create two variables — heads and tails — and set both to 0 when the green flag is clicked. Make both variables visible on screen.
2
Create a third variable called flip. Use a repeat 10 times loop. Inside the loop:
  • Set flip to a random number between 1 and 2
  • If flip = 1: add 1 to heads, say "Heads!" for 0.5 seconds
  • If flip = 2: add 1 to tails, say "Tails!" for 0.5 seconds
3
After the loop finishes, announce the winner:
  • If heads > tails → say "Heads wins!" for 3 seconds
  • If tails > heads → say "Tails wins!" for 3 seconds
  • If heads = tails → say "It's a tie!" for 3 seconds
4
Change the sprite's costume during the loop to match each flip result — one costume for heads, one for tails.
5
Add a backdrop to the stage. For bonus points: play a sound when the winner is announced, and change the backdrop based on who wins.

Correct algorithm (loop runs exactly 10 times, random flip, correct head/tails counting, all 3 winner conditions handled)10 pts
Working program (variables visible, flip/count logic works, correct winner announced)10 pts
Precision (exact say messages, both variables on screen, loop runs exactly 10 times, 0.5s + 3s timing)5 pts
Creativity & efficiency — tiebreaker (costume changes, backdrop changes, sounds, clean code structure)5 pts
